    1. Add the widget to your site Include this script tag before the closing </body> tag on your website:
    2. Submit a pull request Add your site to sites.json in the GitHub repository with your site's URL, name, and description: 
      {
  "url": "https://yoursite.com",
  "name": "Your Site Name",
  "description": "A short description of your site"
}
    3. That's it! Once your pull request is merged, your site will appear in the ring and visitors can discover it through any other member site.

    Customization

    The widget supports theming via data attributes on the script tag:

    Attribute Values Description
    data-theme light (default), dark Built-in color theme
    data-bg Any CSS color Background color override
    data-text Any CSS color Text color override
    data-accent Any CSS color Link color override
